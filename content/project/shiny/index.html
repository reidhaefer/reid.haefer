Shiny is an open-sourced R library for creating interactive applications for data visualization, exploration, and analysis. I have utilized shiny to create web-ready applications for several public sector agencies, which allow users to explore and manipulate data with great flexibility and ease. These applications often include both spatial and tabular data that are combined through a seemless data pipeline that is fully transparent and reproducible. Code for all the apps below can be found on github.

Shiny Examples:

  • User Generated Population Model
  • Interactive Geospatial Zoning Tool
  • Using Big Data For Transportation Planning
  • Explore Local Bike/Ped Data
  • User Submitted Project Information
