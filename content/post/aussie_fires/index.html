During the end of 2019, wildfires proliferated across Australia. As part of the R For Data Science TidyTuesday series, I analyzed Australian temperature and rainfall data to attempt to discern potential contributers to the historic and devastating fires. The visualizations below were created using R. All code can be found on github here.

Are Temperatures Increasing?
